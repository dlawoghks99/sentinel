# Sentinel

실시간 장애 / 지연 감지 모니터링 시스템

---

## 프로젝트 소개
Sentinel은 서비스에서 발생하는 로그를 수집하고  
에러 및 지연 로그를 분석하여  
문제를 빠르게 파악할 수 있도록 도와주는 시스템입니다.

---

## 기술 스택

### Backend
- Spring Boot
- JPA
- PostgreSQL

### Frontend
- React
- Vite

---

## 주요 기능

- 로그 저장 API
- 전체 로그 조회
- 느린 로그 조회
- 에러 로그 조회
- 서비스별 조회
- 통계 API
- 대시보드 UI

---
